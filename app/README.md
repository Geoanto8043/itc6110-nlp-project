# BBC News Assistant

**ITC6110 NLP Group Project** — End-to-end NLP pipeline on the BBC News dataset.

## Features
- **Article Classifier** — paste any news text and get a category prediction with confidence scores
- **Sports Q&A (RAG)** — ask any question; the system retrieves relevant BBC articles and generates an answer using Flan-T5
